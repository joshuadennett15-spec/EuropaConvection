# Docs Index

Top-level documentation is organized as:

```text
docs/
|-- research/     Analysis notes, literature reviews, result writeups
`-- superpowers/  Planning/spec documents captured during implementation work
```

Conventions:

- Put cross-project research notes in `docs/research/`.
- Put implementation plans in `docs/superpowers/plans/`.
- Put design/spec notes in `docs/superpowers/specs/`.
- Keep generated LaTeX sidecar files out of version control.
